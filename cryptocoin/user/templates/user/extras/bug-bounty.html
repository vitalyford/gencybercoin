GenCyberCoin has a bug bounty program! This program will reward you with coins if you find bugs (problems and errors) on the website.

Try to do something that the website would not expect a normal user to do. When you submit something on the website, the information will be processed by the server and the result will be shown to you on the page. What if you send something that a user would usually never think to send? Maybe you can find a place where you can request a certain type of information from the server or you can try to tinker around on the page itself. If you successfully find a bug, you will see a message explaining that type of bug and you will also receive a reward!

Bug Bounty hunting is a real practice through which people can receive monetary rewards on websites such as HackerOne and BugCrowd. It can be quite profitable to participate in these "ethical hacking" Bug Bounty programs. For example, a 10-year-old was rewarded $10,000 by Facebook in 2016 for finding a serious bug on Instagram. Click here to read about it.

ALWAYS think about ethics first when performing a cyberattack. Performing unethical cyberattacks can result in heavy penalties such as prison time and large fines. An example is MalwareTech's case.

Bruce Schneier, a well-known cryptographer, wrote a short essay on hackers and how they can be good or bad. Click here to read it. If you are interested in checking out some real top-level security investigations, drop by KrebsOnSecurity.
